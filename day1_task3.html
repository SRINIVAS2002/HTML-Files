About JS

What is JS? 

JS stands for Java Script.
JavaScript is a programming and scripting language widely used in web development which will enable the dynamic and interactive user experiences.

Who is the creator of JS? 

JS was created by Brendan Eich

When this JS was started creating? 

In 1995, Brendan Eich was invented this JS.

What are the different ways to use JS? 

There are mainly two ways to style HTML.
1) Internal --> writing code in HTML file with the help of script tag either inside the body tag or head tag.
2) External --> separate JS file creation and linking it with HTML file using script tag with 'src' attribute for path specifying.

What are the different uses of JS? 

1) JS can change the HTML content.
2) JS can change the HTML attribute values.
3) JS can change the HTML styles.
4) JS can hide the HTML elements.
5) JS can show the HTML elements.

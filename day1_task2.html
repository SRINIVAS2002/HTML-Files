About CSS

What is CSS? 

CSS stands for Cascade Style Sheets.
It is said to be a language used for styling the document of HTML.

Who is the creator of CSS? 

CSS was created by W3C(World Wide Web Consortium) with significant contributors from Hakon Wium Lie and Bert Bos.

When this CSS was started creating? 

In 1994, Hakon Wium Lie proposed this CSS concept in order to solve presentation problems on web pages.

Where we can do CSS program writing and where we can see its output? 

The programming of CSS language can be done either within the HTML file or else externally creating CSS file and then linking with HTML file through link tags.

Why we need to use CSS? 

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

When did the first version of CSS released?? 

The CSS1 was the first version released in the year of 1996.

What is the latest version of CSS using at present? 

CSS3 was the latest version of HTML currently using.

What are the key features of CSS? 

1) Providing color, layout, font and border properties.
2) Controlling layout of multiple pages at once.
3) Seelctors are used for styling HTML elements.
4) It is adaptable for any kind of device.

What are the advantages of CSS? 

1) Style can apply through any site consistently.
2) Simplifies both website development and maintenance as a change of a line code affects whole site.
3) It will help to form spontaneour and consistent changes.
4) It reduces the file transfer size.
5) It helps to determine changes with in web elements visible on the page.

What are the disadvantages of CSS? 

1) Creates confusion among the different versions of CSS.
2) It might not able to work on another browser if we are working in a particular browser sometimes.
3) There might get cross browser issues while using CSS.
4) Without compatibility confirmation after changes made, there will be affect cause to browser.

What are the different ways to use CSS? 

There are mainly three ways to style HTML.
1) Inline --> which add styles directly to HTML elements with 'style' attribute.
2) Internal --> place styles with in style tag in HTML file within head section only.
3) External --> separate CSS file creation and linking it with HTML file using link tag.
